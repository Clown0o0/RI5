# RI5 - Classes a partir do diagrama UML

Atividade de implementação das classes conforme o diagrama UML (Empresa, Funcionário, Endereço e Telefone).

## Arquivos do projeto
- `telefone.js`
- `endereco.js`
- `funcionario.js`
- `empresa.js`
- `descritorEmpresa.js`
- `teste.js`

## Como rodar

1. Coloque todos os arquivos na mesma pasta.
2. Abra o terminal **dentro dessa pasta**.
3. Execute os comandos abaixo:

```bash
npm init -y
echo '{"type": "module"}' > package.json
node teste.js
